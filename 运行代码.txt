修改_config.yml和_pages/about.md文件以修改内容

然后运行：

git add .                                   # 将修改后的文件添加到Git暂存区
git commit -m                         # 提交暂存区的更改到本地仓库
git push origin master            # 将本地仓库的更改推送到远程GitHub仓库

即可把修改后的文档传到githun仓库里面，然后过几分钟刷新主页即可。

远程仓库地址：https://github.com/MaoMaoLYJ/MaoMaoLYJ.github.io
